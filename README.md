# VoxBoard

**VoxBoard** is a free and open-source keyboard for Android 8.0+ devices. It aims at being modern, user-friendly and customizable while fully respecting your privacy. Currently in beta state.

Forked from [FlorisBoard](https://github.com/florisboard/florisboard) v0.5.2.

## Features

- Clean, modern Material You design
- Glide typing (swipe to type)
- Clipboard manager
- Multiple keyboard layouts (QWERTY, QWERTZ, AZERTY, etc.)
- Emoji panel with search
- Extensive theming engine
- Multi-language support
- Completely offline — no data leaves your device
- No internet permission required

## Building from Source

### Prerequisites

- Android SDK 35+
- JDK 17+
- Gradle 8.x

### Build

```bash
./gradlew assembleDebug
```

The debug APK will be at `app/build/outputs/apk/debug/`.
