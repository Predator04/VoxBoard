# VoxBoard

**Open-source Android keyboard with on-device voice, deep theming, swipe typing, and privacy-first design.**

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

## Features

- 🎤 **Voice input** — On-device Whisper transcription (100% offline, no data leaves your phone)
- 🎨 **Deep theming** — Custom backgrounds, gradients, per-key colors, fonts, animations
- 👆 **Swipe/glide typing** — Gesture-based input alongside traditional tapping
- 🔮 **Smart predictions** — Context-aware word prediction and autocorrect
- 🔒 **Privacy-first** — Zero internet permission, everything on-device, fully open source
- 🌙 **Material You** — Dynamic color theming from your wallpaper (Android 12+)
- 🌍 **Multi-language** — Full support for simultaneous language typing

## Status

**Early development** — not ready for daily use. Follow for progress.

## Building

```bash
git clone https://github.com/Predator04/VoxBoard.git
cd VoxBoard
./gradlew assembleDebug
```

APK will be at `app/build/outputs/apk/debug/`
